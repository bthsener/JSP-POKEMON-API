# JSP-POKEMON-API
![randompokemon](https://user-images.githubusercontent.com/27796975/235181013-68190665-afc4-4984-b836-0b2599e2d8d4.jpg)  
A Java JSP project that displays a random Pokémon using the Pokémon API. This project uses the GSON library for JSON parsing.  

## Installation
```
git clone https://github.com/bthsener/JSP-POKEMON-API.git
```
Open the project in your favorite Java IDE.  
Run the project on a server like Apache Tomcat or GlassFish.  
Visit the homepage at http://localhost:8080/JSP-POKEMON-API.

## Credits
This project uses the [Pokémon API](https://pokeapi.co/docs/v2) and the [GSON](https://github.com/google/gson).
